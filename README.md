# 衆議院選挙区ガイド

2026年衆議院選挙を楽しむための、都道府県別・選挙区別ガイド。選挙情勢だけでなく、各地域の地理・歴史・名所も含めた旅行にも使える情報を整理している。

## ディレクトリ構成

比例代表ブロックごとにフォルダを分け、その中に都道府県のMarkdownファイルを配置している。ディレクトリ名・ファイル名はローマ字表記を使用（GitHub等での互換性のため）。

```
election-docs/
├── README.md
├── 01_hokkaido/
├── 02_tohoku/
├── 03_kitakanto/
├── 04_minamikanto/
├── 05_tokyo/
├── 06_hokuriku_shinetsu/
│   └── 18_fukui.md          # 福井県
├── 07_tokai/
├── 08_kinki/
├── 09_chugoku/
├── 10_shikoku/
└── 11_kyushu_okinawa/
    └── 47_okinawa.md        # 沖縄県
```

## 各ファイルの構成

各都道府県のドキュメントは以下の構成で記述している。

1. **基本情報**: 県の特徴、歴史、政治的傾向
2. **選挙区ごとの情報**:
   - 地域構成と各地域の特徴（観光地、産業、文化など）
   - 2024年選挙結果
   - 2026年選挙の構図と注目ポイント
3. **テーマ解説**: その県特有の政治的争点があれば解説

## 凡例

- 選挙区番号は都道府県コード（JISコード）を使用
- 候補者情報は2026年1月時点のもの
- 「中道改革連合」は2026年1月に立憲民主党と公明党が結成した新党
